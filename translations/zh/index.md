# 太平天国运动 / Taiping Rebellion

太平天国运动是一场争夺中国的战争，一个宗教王国，一次国家建设的实验，也是十九世纪最惨重的灾难之一。它发端于广西的丘陵与市镇，顺长江而下，攻克南京，迫使清廷围绕地方团练重建其军事体系。

太平天国领袖许诺建立一个由天父统治的世界，以被称为天京的新圣城为中心，涤除满清统治、偶像崇拜、鸦片、娼妓与社会不平等。清廷则以经制之师、地方团练、省级统帅——最终以上海及长江下游一带的外国支持武装——予以还击。

这场战争导致数百万人死亡或流离失所，摧毁了城镇与乡村，改变了清朝政治，并为后世的中国政治运动提供了一个可资利用又饱受争议的历史遗产。后来的论者中，有人将太平天国视为农民革命者，有人视之为基督教异端、反满叛军、国家建设者、地方破坏者或悲壮的乌托邦主义者。本维基兼容并蓄这些阐释，但主线叙事围绕的是人物、决策、制度、战役及其后果展开。
