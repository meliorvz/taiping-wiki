# 专题论文：太平天国是什么？

太平天国运动无法干净利落地归入任何一个单一类别。它在经文来源上是基督教的，在语言和政治形式上是中国的，在期望上是千禧年的，在抱负上是改朝换代的，在言辞上是反满的，在招募上主要是农民的，在某些后来的诠释中是革命的。不应让任何一个标签抹杀其他标签。

## 既是基督教又是中国

太平天国借鉴了基督教新教材料——《圣经》（中文译本）、传教士分发的小册子以及洪秀全在广州从罗孝全那里接受的教导。他们的崇拜以上帝为中心，即"天父"；他们守安息日；他们唱赞美诗；他们谴责偶像崇拜，并攻击佛教和道教庙宇，视其为魔鬼。这些特征使早期传教士希望太平天国可能成为基督教改信的载体。

但太平天国并没有成为一个传教教会。洪秀全自称是耶稣基督的弟弟——这一主张没有任何基督教派别能够接受。东王杨秀清在精神附体状态下以上帝的口音说话，这种做法让人想起中国附身灵媒传统而非新教崇拜。太平天国选择性地使用《圣经》，通过洪秀全的启示和他们自己的政治需要来重新解释它。

与此同时，太平天国的宗教语言是深具中国特质的。上帝不是一个外来的神，而是中国经典经典中的一个角色——周朝的至高神，在《尚书》和《诗经》中被提到。太平天国的道德语言——孝、忠、真伪崇拜之分——借鉴了中国传统。洪秀全的天王身份既是一个神学主张，也是对中华神圣君主制传统的重申，以对抗清朝的异族（满人）统治。赖利认为，太平天国通过否定皇帝作为天与地之间中介者的角色，并声称这一角色归洪秀全本人所有，挑战了帝国的神圣秩序。[^reilly][^yuandao]

## 既是农民起义又是建国事业

太平天国从承受压力的人群中招募：贫困农民、失业矿工、流离河工、少数民族、妇女和其他边缘群体。该运动对一种新秩序的承诺——无地者有地，流离失所者有社区，信徒得救——在那些由于经济变迁、人口压力和地方暴力而生活受到干扰的人中间引起了共鸣。太平天国基层士兵的社会构成为这场运动被视作农民起义提供了合理的主张。

但太平天国不仅仅是穷人的起义。他们在南京建立了首都，组织了有精心仪式的朝廷，仿照帝国官僚体制设立了六部（六官），印制了诏书和宗教文本，铸造了货币，并颁布了历法。他们以正式的指挥结构组织军队，有级别、饷级和军纪规定。《天朝田亩制度》和洪仁玕的《资政新篇》提出了全面的社会和政治纲领。这些都是国家的行为，而非暴民的行为。[^acs-jintian][^tianmu][^zizheng]

这场运动的农民基础与其建国抱负之间的张力，是理解其发展轨迹的关键之一。太平天国可以动员农民，但不能一贯地治理领土。他们可以赢得战斗，但不能建立持久的基层行政管理。清廷国家尽管有种种弱点，但在地方社会中有更深的根基——曾国藩和湘军通过团练制度激活了这些根基。[^kuhn]

## 既是反满又超出反满

反满语言弥漫于太平天国的修辞之中。满人是妖魔（妖魔），是篡位者，是崇拜上帝真正信仰的玷污者。清朝皇帝是坐在窃取来的宝座上的"妖子"。这种语言服务于动员汉人反对王朝，并将战争框定为一神圣的斗争而非政治叛乱。

但目标是超出族群的。在太平天国的神学中，满人不仅仅是异族统治者，还是与天序为敌的妖力代理人。抗清的战争是一场抗妖的战争，一场政治服从于神命的宇宙斗争。这一神学维度将太平天国的反满主义与后来革命者的世俗民族主义区分开来。孙中山可以援引太平天国作为反满的先驱，但他不能采纳他们的神学。[^reilly]

## 最佳简短定义

太平天国是一个以宗教授权、反抗清朝的政权和军队，它源自华南的地方危机，借鉴了新教基督教和中国的宗教资源，大量招募了来自社会边缘的人群，为争夺帝国控制权发动了一场持续十四年、杀死数百万人、并永久改变了清廷国家和中国社会的内战。

## 争议

分类辩论是太平天国研究中最持久的之一。这场运动最好被理解为宗教、政治叛乱、社会革命还是某种组合？答案部分取决于历史学家在问什么问题，部分取决于哪些资料被优先考虑。一位关注太平天国文本的历史学家会强调宗教；一位关注招募的历史学家会强调社会构成；一位关注清廷反应的历史学家会强调军事和制度变革。最好的学术承认多种视角的有效性，同时认识到没有单一框架能捕捉全貌。
