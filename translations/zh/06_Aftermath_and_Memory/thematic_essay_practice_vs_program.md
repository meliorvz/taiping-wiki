# 专题论文：太平天国实际做了什么？

太平天国的公告告诉我们领导人想要什么。占领记录、地方记述和物质证据告诉我们发生了什么。好的历史在不把读者淹没在附加说明中的情况下，将这两个层面分开。纲领与实践之间的差距是太平天国研究中最重要的课题之一，它需要仔细关注每一种能够回答相应问题类型的资料。[^michael]

## 纲领

太平天国产生了异常丰富的纲领性文本。《天朝田亩制度》（《天朝田亩制度》，1853年）承诺土地共同耕作、公共粮仓、每个社区设立礼拜堂、按人口和土质分配土地，以及由官方监督生产和分配。它设想了一个政教一体的土地秩序，在这个秩序中，国家——即太平天国朝廷及其神圣君王——直接管理经济生活。该文本关于公共财产和平等分配的语言使其吸引了后来的激进读者，但牟复礼的文献研究表明，它是作为一份宣言发布的，而非作为一套行之有效的行政制度来实施的。[^tianmu][^michael]

洪仁玕的《资政新篇》（《资政新篇》，1859年）提出了一种不同类型的纲领：发行报纸传播信息、设立邮政局、建设铁路和轮船、设立银行和保险公司、与外国开展外交接触，以及鼓励商业和工业。洪仁玕是太平天国领导人中最具国际视野的，他的建议反映了他通过传教士接触和他在香港的居留所接触到的西方制度。但《资政新篇》也基本上是未实施的——洪仁玕的影响力受到朝廷政治、持续的军事危机以及洪秀全的宗教绝对主义的限制，后者使其难以采用看似借鉴外国（因此可能异端）模式的政策。[^zizheng][^michael]

太平天国还颁布了历法、铸造了货币，并建立了一个以《圣经》和太平天国文本而非儒家经典为基础的正式科举制度。这些制度成果显示太平天国有建国的抱负，但它们在南京以外实施的程度上差异极大。

## 实践：南京朝廷

南京朝廷是太平天国国家表达最充分的形式。洪秀全作为天王在原来的明朝宫殿群（重建后改名为天王府）中执政。朝廷拥有精美的仪礼——早朝、仪式行列、规范化的称谓形式——结合了中国帝国实践的特征与太平天国的宗教创新。六部（六官）运作，尽管其有效性取决于控制它们的各王的性格和竞争。

但南京朝廷也是这场运动最具破坏性的内部冲突的场所。1856年的天京事变——清洗杨秀清及其追随者，随后诛杀韦昌辉，以及石达开的被排挤——证明太平天国国家是一个人独裁政权，其制度无法解决领导人之间的冲突。维系这场运动的宗教权威（杨秀清以灵媒附体代表天父说话）成为争议的来源，当这一权威受到质疑时。1856年以后，朝廷从未完全恢复其凝聚力，战争的后期以派系斗争和中央控制力的下降为特征。[^michael][^luo]

## 实践：战场上的军队

太平军是在因战区不同、时期不同而变化巨大的条件下作战的。在早期（1850—1853年），军队快速移动，就地取食，压倒了组织不善、领导无力的清军。南京建立后，太平军同时在多个战区行动——北伐、西征、向上海的东征——中央协调程度各不相同。

战场上的军队需要粮食，他们通过组合从占领区征用、预先储备的物资以及（有时是强制性地）从当地居民购买来获得。军纪各有不同：一些太平军将领对他们的部队保持严格控制；另一些人统率的军队行为上与匪徒无异。清方资料一贯将太平军描述为贪婪劫掠，但这些记述服务于宣传目的，应谨慎对待。太平天国资料将军队呈现为纪律严明的神圣战士，这服务于不同的宣传目的，也应谨慎对待。[^michael]

## 实践：占领区

太平天国占领的经历因地点而异、差别极大。一个在长期内处于太平天国稳定控制下的城市——如江西和安徽的一些城镇——可能经历类似于常规行政的状态：税收征收（税率不同且文献记载不足），任命地方官员（通常是加入太平天国的本地人），压制传统宗教实践（庙宇被毁或关闭，地方神灵被压制），以及推行太平天国的宗教和社会规定（男女隔离、守安息日、禁止鸦片烟酒）。

一个被一支行进的太平军经过的村庄可能只经历征用和离开。一个位于太平军与清军之间前线上的县，可能经历双方的反复占领，百姓在双方争夺粮食、劳力和忠诚之间左右为难。差异如此之大，以至于没有任何一种关于"太平统治"的描述能适用于所有情况。

太平天国对庙宇和宗教造像的攻击——梅尔清等人所记录的毁像——是太平天国占领最具一贯性的特征之一。佛教和道教庙宇的造像被剥去，地方神灵雕像被摧毁，传统民间宗教空间被改为他用或留在废墟中。这种毁像表达了真实的太平天国宗教信念——崇拜上帝要求摧毁假神——但也导致其失去当地人的民心，因为他们的宗教生活正是以这些制度为中心。黎民百姓的许多地区中，清廷的收复是由其庙宇和神灵曾遭到太平天国攻击的地方精英促进的。[^meyer][^kuhn]

## 如何阅读太平天国政策

当阅读一项太平天国政策时，提一系列区分不同证据层次的问题是有益的：这是在公告中承诺的吗？这是在给下属官员的指令中下令的吗？这是实际执行过的吗——如果是，在哪里、持续了多久？实施的证据是来自太平天国资料、清方资料、外国资料还是战后编纂的地方方志？是出于敌人之口用于论战目的吗？是通过物质遗存（建筑、钱币、碑文）在本地观察到的吗？是通过带有特定解释目的的历史学家在后来论证的吗？

答案通常因政策不同而不同。《天朝田亩制度》是公告的，不是系统实施的；礼拜天的安息日被广泛遵守；鸦片禁令以不同严厉程度执行；男女隔离在驻军中适用，但在占领区不那么一致。承认这些区分比要么将所有太平天国政策视为已实施、要么将所有太平天国宣言视为空洞修辞能产生更准确的图景。

## 争议

历史学家争论太平天国占领在多大程度上构成真正的治理，还是带有薄薄行政外衣的军事占领。证据因地点和时期而异，这一争论不太可能在一般层面得到解决——它需要能显示太平天国在特定县镇实际做过什么的地方研究。

一个相关的争议涉及太平天国毁像的有效性。摧毁庙宇和造像是成功地重新塑造了民间宗教实践，还是产生了助长其失败的抵抗和怨恨？梅尔清关于战后庙宇重建的著作表明民间宗教在战后恢复了，但太平天国毁像对宗教信仰（而不是宗教基础设施）的影响程度是一个更难回答的问题。[^meyer]
