# 史料指南

本页面面向编辑者和进阶读者。主体条目不应被出处警示打断叙事，除非某一争议确实影响了历史叙述本身。

## 史料族系

### 太平天国印书（印书）

太平天国政权在天京（南京）设有自己的印刷机构，刊行宗教、行政、军事文本的官方版本，并在太平天国控制区域内广泛分发。

现存情况：已知存世约 40—45 种，但其中有些仅存寥寥数册。主要藏品见于大英图书馆、法国国家图书馆、美国国会图书馆，以及南京图书馆和中国国家图书馆等多家中国机构。

查找途径：
- 《太平天国印书》现代影印本，部分收录于《民国丛书》第二编，可在 Internet Archive 上获取。尚未在开放数字资源库中找到完整版本。
- Michael 著作第二至三卷（1971）翻译了约 400 份太平天国文件，其中包括主要的官方文本。
- 中文维基文库收录了数份关键文献的转录文本（《天朝田亩制度》《资政新篇》《原道救世歌》）。
- Internet Archive 上有若干单行本的影印件（如《英杰归真》，26.4 MB PDF）。

注意事项：太平天国印书属于宣传文献。它们教导的是领导层希望臣民相信的内容，而非实际发生之事。许多文献在不同版本之间有所修订；1853 年之前的版本有时在神学观点上与 1853 年之后的版本存在差异。

### 清朝奏折与上谕

清朝的档案制度是前现代世界规模最大、体系最严密的制度之一。官员向皇帝呈递题本（例行）或奏折（秘密），皇帝则通过朱批或由军机处颁发的正式上谕作出回应。

档案结构：
- 《清实录》：各朝皇帝去世后编纂的编年摘要。它们浓缩而非照录奏折原文。可用于了解官方叙事；但会对政治上敏感的内容加以删削、润饰或改写。
- 《清政府镇压太平天国档案史料》（26 卷，1990—2001）：最大规模的清代镇压太平天国档案文献汇编，来源于北京中国第一历史档案馆。尚未在开放数字资源库中获取。这是当前史料清单中最为重大的缺环。
- 中国第一历史档案馆门户网站（https://fhac.com.cn）：提供目录检索界面。部分数字化文献可访问。全文访问可能需要机构授权。

奏折所记录的内容：兵力调动、后勤、战报、请赏请罚、地方状况、太平天国被俘人员的审讯记录、任命推荐。

奏折所省略或歪曲的内容：官方战败始终被轻描淡写。敌方兵力常被夸大，以使胜利更显赫或为使增援请求合理化。太平天国信仰的族群与宗教维度完全以清朝意识形态的话语来表述（粤匪、长毛、教匪）。

### 地方志（地方志）

中国地方志是一种独特的历史文类：系统编纂一地的地理、行政、人口、经济、人物以及重大事件。

所记录的内容：
- 县、府边界与行政沿革
- 人口与地丁银册（尽管数据常抄录自早期版本）
- 地方贤达传记（《忠义》《孝友》《列女》等门类），常包含太平天国战争期间死亡的详细记述
- 军事（兵事/武备门类）——地方战事编年
- 地方机构（学校、祠庙、仓廒、桥梁）及其毁坏与重建
- 忠义录，列名在战事中丧生的本地人

所省略的内容：
- 对太平天国治理的任何正面或中性描述
- 太平天国行政结构（除非作为破坏证据而列出）
- 民众对太平天国的支持
- 广东、广西冲突中的族群维度（客土之争）
- 经济破坏常被轻描淡写，以恢复地方赋税定额

编纂偏向：方志由地方绅士在知县监督下编纂，通常距事件发生已有一代或更久。编纂者几乎清一色来自曾反对太平天国且从清朝恢复中获益的家族。战后方志往往强调破坏、殉难与效忠朝廷，以此作为地方社会道德重建的一部分。

现有馆藏：安徽通志（1200 余卷）、江西通志（约 90 卷）及福建通志（部分）可在 Internet Archive 上获取。广西、湖南、湖北、江苏、浙江、四川等省通志尚未在开放数字资源库中找到。

### 传教士记录

有关太平天国的传教士资料十分丰富，但需要仔细滤除教派偏见。

教派格局：
- **英国新教传教士**（主要为伦敦会、大英圣公会）：人数最多，记录最为详尽。伦敦会传教士（W. H. Medhurst、Walter Henry Medhurst Jr.、William Milne、Joseph Edkins）与太平天国的接触最为持久。其记录藏于伦敦大学亚非学院图书馆和耶鲁大学神学院图书馆。
- **美国新教传教士**（主要为美部会、南方浸信会）：罗孝全（I. J. Roberts）是唯一在太平天国起义前后均见过洪秀全的传教士。其信件和报告藏于南方浸信会历史图书馆与档案馆（田纳西州纳什维尔）。裨治文（E. C. Bridgman）对 1854 年天京之行的观察，在形成美国对太平天国的负面评价方面影响甚大。
- **法国天主教传教士**（主要为巴黎外方传教会）：从最早的报道起，在华天主教传教团便对太平天国持敌视态度。太平天国对天主教圣像及三位一体教义的排斥，加之摧毁天主教堂、杀害传教士的行为，导致了持续不断的敌对报道。法国外交档案（法国外交部档案馆）应作为天主教视角的参考资料。

偏见模式：
- **1860 年前与 1860 年后**：传教士关于太平天国的写作在 1860 年之前总体上较为同情，其时太平天国破坏偶像的性质与规模已无法忽视。从谨慎乐观转向公开谴责，是传教士记录中的一个重要模式。
- **教派滤镜**：公理宗（伦敦会）倾向于以公理会教会治理模式评判太平天国；浸信会（罗孝全）以成人浸礼的浸信会标准评判；天主教则以宗徒继承与圣事神学的天主教标准评判。这些框架均未能以太平天国宗教自身的方式理解太平天国宗教。
- **语言障碍**：极少有传教士具备阅读太平天国神学文本原文的文言文素养。大多数依赖翻译人员、摘译或二手报告。

### 条约口岸报刊

《北华捷报》（North-China Herald），1850 年起在上海每周出版，是关于太平天国最重要的英文报刊资料。它实际上是上海外商社群的非官方喉舌，但也刊载中国官方文件、太平天国文告的翻译，以及外国领事与清朝官员之间的通信往来。

有关太平天国的报道：
- **1850—1852**：广西乱事的早期报道，主要来自传教士和商人的通信
- **1853**：南京陷落的广泛报道；在上海发现的太平天国文告的翻译；太平军进逼上海的报道
- **1854—1859**：北伐及天京事变期间的零星报道；亚罗号战争（1856—1860）的报道日益占据主导
- **1860—1863**：太平天国报道的高峰期；苏州、杭州陷落；外国干预之争；华尔、白聚文及戈登的战役；苏州杀降；李秀成在江浙的战役
- **1864**：南京陷落；曾国藩的报告；战后善后

《北华捷报》的长处：来自上海外国人社群近乎实时的报道；通过条约口岸渠道流传的中国文件的翻译；具有军事含义的详细商业航运情报；身处内地的传教士的来信；外国人社群舆论的形成及其对英国政策的影响。

《北华捷报》的短处：清朝方面的战争（接触清朝官员的渠道有限）；超出条约口岸情报网络覆盖范围的内地战役；太平天国方面（至 1860 年该报已一致敌视太平天国）；任何未由其通讯记者翻译的中文资料。

现有馆藏：约 3,900 期（1850—1941）可在 Internet Archive 上获取，含部分 OCR 文本。已识别出 1850—1864 年间约 190 期。全套馆藏的关键词全文索引尚未完成。

## 史料状态标识

| 标识 | 含义 |
|---|---|
| 直接查阅 | 在本次构建中已核对过文本、目录或官方记录。 |
| 已识别但未直接查阅 | 已知史料，尚未阅读。 |
| 高优先级获取目标 | 深入撰写条目所需。 |
| 存疑或有争议 | 文本流传、署名或真实性需特别谨慎对待。 |
| 因不可靠而排除 | 仅可用于接受史研究或作为线索。 |

## 史料类型分类

| 类型 | 示例 | 权威性 |
|---|---|---|
| 太平天国官方文献 | 《天朝田亩制度》《资政新篇》《原道救世歌》 | 领导层的宣称与命令 |
| 清朝档案 | 奏折、上谕、《清实录》 | 官员的报告、决策与宣称 |
| 当代外国人叙述 | Lindley、Callery & Yvan、Scarth、Meadows | 目击者视角，带有国别/翻译者偏见 |
| 外交/政府文件 | BPP、US FRUS、法国外交档案 | 外国政府官方视角 |
| 条约口岸报刊 | 《北华捷报》 | 上海外国人社群视角，近实时报道 |
| 地方志 | 省、府、县志 | 地方绅士记忆，战后重建文献 |
| 中华人民共和国学术研究 | 罗尔纲、茅家琦、刘晨 | 马克思主义/民族主义史学框架 |
| 西方学术研究 | Michael、Kuhn、Reilly、Meyer-Fong、Spence、Kilcourse | 不同分析框架 |
| 台湾/香港学术研究 | 简又文、沈云龙 | 非马克思主义中文史学 |
| 民国时期史学 | 罗尔纲 1937、Hail 1927 | 早期学术论著 |
| 传教士记录 | 伦敦会档案、罗孝全通信 | 宗教视角，带有教派偏见 |

## 实用原则

将史料不确定性放入注释、史料页面或争议页面。保持面向读者的叙事可读。
