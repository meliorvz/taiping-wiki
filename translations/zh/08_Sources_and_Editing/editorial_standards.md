# 编辑规范

## 面向读者的文字

主条目应为感兴趣的读者而写。首先讲述事件。将技术性的史料问题置于注释和编辑页面中，除非该问题改变了故事的含义。

不要在正文中写入对编辑说话的语句。不要在叙事页面中写入"使用此术语"等字样。命名规则应放在本页面说明。

## 脚注

使用 Markdown 脚注。不得在句子中间插入原始引文评论。

示例：

```markdown
太平军于 1853 年 3 月攻克南京，并将其更名为天京。[^michael1]

[^michael1]: Franz Michael, *The Taiping Rebellion: History and Documents*, Vol. I: History (Seattle: University of Washington Press, 1966)。
```

### 脚注规则

1. 面向读者正文中的每一项事实性陈述，都必须可追溯至一个已标识的史料。如果没有史料支持某一陈述，则用 `[citation needed]` 标注，或将其移至草稿/笔记页面。
2. 使用最直接的史料。如果 Michael 第二至三卷提供了某份文件的译文，则引用 Michael，而非对该译文的二手概括。
3. 不得引用百科全书或其他三级史料。每项陈述都必须可追溯至原始史料或专业的二手史料。
4. 当某一史料存在争议或已知偏见时（例如 Lindley 的亲太平天国党派立场、清廷奏折夸大敌兵数量的倾向），应在脚注中注明此局限性。
5. 中文史料以原文汉字辅以拼音罗马化方案引用。首次脚注引用时提供书名的英文翻译。后续引用可仅使用英文标题。
6. 史料获取信息（URL、LibGen ID、Internet Archive 标识符）应在首次脚注引用时包含在内。
7. 脚注中不得使用"op. cit."或"ibid."。应使用简短标题引用（如"Michael, History and Documents, Vol. I, p. 45"）。
8. 支持同一陈述的多条引证应合并在同一条脚注中，以分号分隔。

## 中文术语

在主要页面首次使用时，按以下格式写出：汉字、拼音、翻译。

示例：天京，Tiānjīng，"Heavenly Capital"。

首次使用后，使用最适合该页面的英文术语或拼音形式。

### 中文段落的翻译规范

1. **三级翻译规则**：当面向读者的正文中引用中文段落时，应提供：
   - 中文原文（置于脚注中，除非段落足够简短可放入行内）
   - 直译（置于脚注中，尤其当措辞存在争议时）
   - 润色翻译（置于正文中，以保证可读性）

2. **音译**：所有中文术语使用汉语拼音。不得使用威妥玛拼音，除非引用的史料本身即使用威妥玛拼音（在此情况下，首次使用时应在括号内附注拼音对应形式）。

3. **人名**：正文中所有中国历史人物均使用中文姓名顺序（姓在前）。示例：Hong Xiuquan，而非 Xiuquan Hong。对于以西方姓名形式更为人所知的人物，首次使用时注明其中文原名。

4. **官衔与职务**：以描述性方式翻译官职（如将"大学士"译为"Grand Secretary"），而非使用罗马化汉语。如果太平天国特有的官衔没有对应的英文，首次使用时使用拼音加翻译（如"Liang Sima 两司马, 'two-sima officers'"）。

5. **地名**：正文中中国地名使用标准拼音形式（如"Guangzhou"而非"Canton"，"Beijing"而非"Peking"）。相关时在首次使用时以括号注明历史名称。

6. **术语一致性**：一旦为关键术语选定翻译（如以"Heavenly Kingdom"译"天国"，以"Heavenly Capital"译"天京"），应在所有页面中一致使用。不得为追求文采变化而变换翻译。

7. **不翻译的术语**：某些术语应保留拼音：
   - 特定机构的专有名称（如"博济会"译为 Boji Hui）
   - 翻译存在严重争议或带有误导性含义的术语
   - 拼音形式在英文学术语境中已通用的术语

## 史料分类规则

本维基引用的所有史料均按以下方案分类。分类应出现在史料数据库中，供编辑者在评估史料可靠性时参考。

### 按来源分类

| 类别 | 描述 | 可靠性说明 |
|---|---|---|
| 太平天国官方文献（T） | 太平天国政权产生的文件 | 记录的是宣称，而非事实；在意识形态和自我表现方面极有价值 |
| 清朝档案文献（Q） | 奏折、上谕、清朝官方记录 | 记录的是官员的报告；存在向上扭曲 |
| 当代外国人记述（C） | 非中国观察者的目击记述 | 对中文史料所缺乏的细节有独特价值；受国别、语言及文化偏见影响 |
| 条约口岸报刊（P） | 在条约口岸出版的报纸和期刊 | 近乎实时的报道；对内地及中文史料的接触有限 |
| 地方志（G） | 省、府、县志 | 绅士视角；战后编纂；覆盖面系统，但有可预见的沉默 |
| 现代学术研究——西方（W） | 英文及其他西方语言的学术著作 | 方法学质量参差不齐；需逐一评估 |
| 现代学术研究——中华人民共和国（PRC） | 中华人民共和国产生的学术著作 | 马克思主义框架；1950 至 1970 年代存在较强体制偏见；1980 年代以来趋于细致 |
| 现代学术研究——台湾/香港（TW） | 台湾及香港的学术著作 | 非马克思主义；可接触民国时期档案 |
| 传教士记录（M） | 新教及天主教传教团的记录 | 在宗教问题上有价值；存在较强的教派偏见 |

### 按获取状态分类

| 状态 | 定义 |
|---|---|
| 已获取 | 已获得数字副本并本地存储 |
| 已直接查阅 | 已阅读或查阅文本（网络或本地） |
| 已识别，尚未查阅 | 已知其存在并已定位，尚未阅读 |
| 已识别，可借阅 | 可在 Internet Archive 或类似平台上获取，需借阅/认证 |
| 部分获取 | 部分卷册或章节已获取，其他未获取 |
| 未找到 | 在目录或参考文献中已定位，但未在开放数字资源库中找到 |
| 尚未搜索 | 已知存在，尚未在数字资源库中尝试搜索 |
| 无数字版本 | 已知仅存实物形式或在机构付费墙之后 |

## 证据处理规则

1. **互证要求**：任何重要的事实性陈述不得仅依赖单一史料，除非该史料是事件的唯一存世记录。当某一陈述依赖单一史料时，应在脚注中注明。

2. **史料三角化**：对于存在争议的事件，应将相关各方的叙述并列呈现。不得综合出一套"平衡"叙述以掩盖分歧。示例：苏州杀降（1863）应将李秀成的叙述、李鸿章的奏折、西方目击者报告以及学术争论作为各自独立的要素呈现。

3. **负面证据**：证据之缺乏不等于缺乏之证据。当某一陈述因史料记录中的已知缺环（如天京陷落后太平天国档案被毁）而无法得到证实，应明确说明。

4. **沉默作为证据**：史料未言之事与史料所言之事同样重要。当某一史料明显省略了如果属实则理应提及的内容，此可作为弱负面证据采纳。

5. **数字与统计**：前现代中国史料中的所有伤亡数字、兵力人数及人口数字均为近似值。整数（尤其是以万为单位的倍数）是惯例性的，而非统计性的。应提供范围（如"在 2000 万至 3000 万之间"）而非点估计（如"2700 万死亡"），并始终注明估算的来源及所使用的方法。

6. **传抄链条**：对于翻译史料，应追溯翻译谱系。如某段太平天国文献的内容仅通过一份引述它的清廷奏折、一份对之进行释义的传教士翻译以及一份现代学术重建文本为世所知——应将三者全部呈现，并解释其中的差异。

7. **引文**：尽可能直接引用源文本。以"……"标示省略；以[方括号]标示编辑插补。不得悄无声息地修改引文以使其配合叙事。

## 日期

叙事中使用公历日期。仅在编年注记或专业页面中添加农历日期。

## 争议日期

如果不同史料给出不同日期，在主叙事中写出事件发生的月份，将确切日期问题置于注释或编年页面中。

## 伤亡数字

概述中使用宽泛估计。仅当附有地域、方法和来源时方可使用确切数字。

## 政治遗产

将相关主张分类为：直接影响、声称的影响、修辞借用、史学类比、后世造神、或未经证实的所谓影响。

## 禁用文风

避免夸大意义的断言、泛泛而谈、标语口号及装饰性语言。写明人物。写明地点。写明文献。使用简短的主题句。

## 质量核查

在条目从草稿进入已发布状态之前，应核实：

1. **史料核查**：每一项事实性陈述均有指向已识别史料的脚注。
2. **术语核查**：中文术语首次使用时均附有汉字 + 拼音 + 翻译；拼音正确（核对声调）。
3. **日期核查**：所有日期均为公历；农历对应日期仅出现在注释中。
4. **人名核查**：所有人名均遵循中文姓前名后的顺序；所有地名均使用标准拼音。
5. **数字核查**：伤亡数字均注明史料与方法；整数被视为近似值处理。
6. **偏见核查**：具有已知偏见的史料（亲太平天国、反太平天国、教派偏见、民族主义偏见）在脚注中予以标注。
7. **翻译核查**：引用的段落同时包含中文原文（在脚注中）和润色后的英文翻译（在正文中）。
8. **链接核查**：所有内部维基链接均可解析；所有外部 URL 均可访问。
9. **陈述核查**：每一项重要陈述至少有两个独立史料互证，或注明其单一史料来源的局限性。
10. **可读性核查**：一个不熟悉中国历史的读者应能理解该条目；所有专业术语均在首次使用时予以解释。
