# 太平天国军事组织

太平军起初是一场宗教运动的武装力量，渐次演变为一个敌对国家的军事基础。其优势在于纪律、机动性、道德规范、指挥层级以及在敌对领土上调遣大量人员的能力。其弱点——守土、持久围城和经受领导层危机等方面的困难——最终证明是致命的。[^michael-ii][^luo-ergang]

## 军事层级

太平天国的军事组织建立在一套将民政与军衔融为一体的嵌套层级之上。田亩制度使用相同的职衔称呼农业官员和军事指挥官，这反映了该运动对于单一神圣-军事秩序的设想。由最小单位至整个军的完整链条为：[^michael-ii][^luo-ergang]

| 层级 | 指挥官 | 编制 |
|-------|-----------|-------------|
| 伍 | 伍长 | 4兵 + 1指挥官 |
| 两 | 两司马 | 5伍 = 25人 |
| 卒 | 卒长 | 4两 = 104人 |
| 旅 | 旅帅 | 5卒 = 525人 |
| 师 | 师帅 | 5旅 = 2,630人 |
| 军 | 军帅 | 5师 = 13,156人[^luo-ergang] |

军是最大的标准编制。一名军帅在纸面上指挥约一万三千人，但实际战场兵力因伤亡、逃亡和招兵而差异极大。在军以上，指挥权由军事诸王行使——杨秀清（东王）、韦昌辉（北王）、石达开（翼王），以及后来的陈玉成、李秀成等将领——以及由朝廷任命的元帅和将军。[^luo-ergang]

两司马，这一最低级军官，是关键环节。他指挥二十五人，对他们个个了如指掌，带领他们战斗和礼拜，并对其纪律、补给和品德状况负责。这一制度的优势依赖于称职的两司马——当无德无能之人担任此职时，其裂隙便显现出来。[^michael-ii]

## 军队规模与编制

对太平军总兵力的估计差异极大。1851年1月金田起义时，该运动约有1万至2万名武装追随者，包括家属在内。到1853年3月太平军抵达南京时，兵力已膨胀到大约10万至20万，但其中许多是挑夫、家眷和被裹挟的平民，而非战斗兵员。在19世纪50年代中期达到顶峰时，太平军可能拥有30万至50万战斗员；合计随营人员和家属则约达50万至60万（罗尔纲估算），分布在各省的多支军队中。[^luo-ergang][^michael-i]

太平天国将兵力编入区域性军队——东王军、北王军、翼王军，以及后来李秀成、陈玉成及其他将领所统率的部队。1858年11月三河之战中，陈玉成从分散的安徽守军中集中了三至五万兵力——骑兵、步兵及辅助部队——对李续宾部六千孤军实施双重包围，全歼该部，歼灭清军将官四百余人，为湘军战史上最惨之一败。这些军队半自主地运作，鉴于距离遥远实属必要，但自主性造成了离心压力，削弱了中央指挥。[^luo-ergang]

## 军纪法规

太平天国维持着成文的军事行为规范。主要文本是《太平条规》和《行军总要》。核心规则要求：[^michael-ii]

- 各层级官兵绝对服从指挥
- 每日敬拜天父
- 禁止劫掠、强奸和擅自杀戮平民
- 军营中男女分营
- 妥善保管武器、粮秣和钱物
- 哨兵勤务、侦察和信号规程
- 对失职哨兵、睡眠值岗和逃兵的惩处

惩戒法规极为严厉。斩首是对严重罪行的标准刑罚——强奸、杀人、劫掠、逃亡、通敌。鞭笞常用于较轻的过失。军官对其部下的行为负有责任。呤唎于19世纪60年代在太平军中服役，他报告称处决是当众迅速执行的，以维持纪律。[^lindley]

法规与实践之间的差距因地而异。在有能将领率领的纪律严明的部队中，纪律据报甚为良好——士兵花钱购买补给、避免与百姓接触、履行宗教义务。在监管不力的部队中、在战役的远翼地带、在强行穿越敌对领土的行军途中，纪律则遭到破坏。清朝的宣传夸大了太平军的暴行；太平天国的宣传则夸大了自身的克制。真相大概介于两者之间。[^luo-ergang]

## 武器、补给与后勤

太平军使用的武器是中国传统武器与西方火器的混合。标准步兵武器是长矛，辅以刀剑、戟和弓。早期的太平军火器极少。1853年之后，太平天国获得了越来越多的火枪、步枪和大炮——有些是从清军武库中缴获的，有些是通过上海的外国商人和走私者购买的，有些是在安庆及其他中心的太平天国作坊中制造的。[^lindley][^michael-iii]

西方观察者指出太平军的火力不均衡。有些部队装备精良，配有现代步枪；另一些部队仍在使用长矛和火绳枪作战。大炮被用于围攻和城防，但太平军的炮兵总体上不如清军于19世纪60年代在欧洲援助下部署的重型攻城炮。[^lindley]

补给是恒久的难题。圣库制度为中央统辖的部队提供口粮——粮食、盐、油和小额薪饷。野战部队就地征粮、征发和购买补给。1853–1855年的北伐显示了在远离补给基地的冬天条件下作战的致命后果：全军覆没，将领被俘处决。[^luo-ergang]

太平天国维持着一个后勤队伍——挑夫、水手、补给军官——通过水路和陆路运送粮食、武器和物资。控制长江及其支流对太平军的后勤至关重要。1861年失去安庆以及清军内河封锁的收紧，切断了天京的补给线，将都城置于饿殍之中。[^michael-i]

## 长江上的水师

太平天国在长江及其连通水道上维持着一支颇具规模的水师。内河运输对于调动军队、运送补给和赋税至为关键。太平军舰队包括数以千计的船只——缴获的清军战船、征用的民船以及专门建造的战船。舰队起源于1852年11月，当时太平军在放弃长沙围城后，于益阳、岳州缴获了数千艘船只，获得了使他们能够抵达武昌并沿长江直下南京的内河机动能力。杨秀清直接关注水师事务，组织了一支在19世纪50年代大部分时间中控制长江中下游的舰队。[^luo-ergang]

太平军水师实力在19世纪50年代中期达到顶峰。舰队在1852–1853年间将军队运送至长江下游，供应天京，并支持了江西和安徽的作战。但太平军在水师的系统化组织方面从未与清军匹敌。清军——在西方援助下——在曾国藩统率下发展了现代化的水师力量即长江水师，逐渐从太平军手中夺取了内河控制权。到19世纪60年代初，清军炮艇和铁甲舰封锁了长江，孤立了太平军的各个据点。[^kuhn][^michael-i]

## 将领与领导层问题

太平军指挥体系是王朝式的和人身依附性的。最初五王——洪秀全、杨秀清、萧朝贵、冯云山和韦昌辉，石达开随后加入——组成了最高指挥部。这一体系将权力集中于一个由共同的广西起源、客家身份和宗教承诺所凝聚的小群体。当冯云山和萧朝贵于1852年战死、杨秀清和韦昌辉在1856年天京事变中丧生后，指挥架构随之坍塌。[^michael-i]

1856年以后的太平军由新一代将领领导——陈玉成（英王）、李秀成（忠王）等人——他们凭借战场功勋而非早期运动成员身份而崛起。这些将领颇具才能，但缺乏最初诸王的神圣权威。他们以相当大的自主权行事，有时与朝廷协调，有时则置若罔闻。指挥的碎片化削弱了太平天国，使协调一致的策略难以实施。[^luo-ergang]

李秀成在其被俘后于1864年写下的著名供词中，提供了晚期太平天国军事行动最为详尽的内部记述。他描述了指挥混乱、补给失败、朝廷干预以及防守日益孤立的天京之不可能。他的记述——如同所有战俘陈述——须以批判态度阅读，但它提供了一幅军事体系崩溃的生动图景。[^michael-iii]
