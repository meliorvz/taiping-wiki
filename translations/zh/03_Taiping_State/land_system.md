# 天朝田亩制度

《天朝田亩制度》是太平天国文献中最为雄心勃勃地描述其计划中社会经济秩序的一篇。该文于1853年末——太平天国定都天京后不久——颁布，承诺按人口分配土地、建立公有仓储、由公共体系赡养弱势群体、推行道德教化、强制执行礼拜以及建立将乡村农业与军事指挥相连的地方官员层级体系。[^tianmu][^michael-ii]

此文至今仍是太平天国研究中学者争论最激烈的一份文献。革命时代的史学家视其为农业共产主义纲领的明证。怀疑论者则指出，从未有证据表明它曾被系统性地付诸实施。问题的关键已不在于田亩制度是否被实际推行，而在于它揭示了太平天国的何种意图，以及它是在怎样的条件下产生的。[^luo-ergang][^michael-ii]

## 九等田制

《天朝田亩制度》制定了一套详尽的农业土地分级方案。天下所有田地被划分为九等——上等三级（上上、上中、上下），中等三级（中上、中中、中下），下等三级（下上、下中、下下）。最高等级的田——上上田——被定义为早晚二季阳光充足下亩产一千二百斤的田地；最低等级——下下田——亩产仅四百斤。每降一个等级，产量预期减少一百斤。[^tianmu][^luo-ergang]

土地按家庭人口分配，不分性别。六口之家得三亩上等田及三亩下等田，良田与瘠田搭配以求公平。五口之家得其五亩；四口之家得其四亩；三口之家得其三亩。凡年满十六岁者得一份成年人全份，十五岁以下者得半份。文本明确规定男女均在配田之列。[^tianmu][^michael-ii]

这一制度从根本上背离了清代的土地占有实践。清代社会建立在私有财产、宗族信托、租佃、雇佣劳动以及承认既成占有的赋税制度之上。田亩制度设想了一个土地不为私人所有、而由神圣国家临时分配的世界——土地随家庭人口的增减而重新分配，剩余产品则汇集到由军官监督的公共仓储中。[^kuhn]

九等田的语言借鉴了经典行政范畴——《周礼》中描述了土地等级——但将其与太平天国的宗教主张融为一体。天父拥有天下土地；国家仅代天父管理之。[^reilly]

## 二十五户制与两司马体系

田亩制度中最具影响力的组织概念是二十五户单位。这一基层社会单元由一名两司马管辖，其职责涵盖农业、仓储、礼拜、教育、纠纷调解、道德监察及军事动员。[^tianmu][^luo-ergang]

每个二十五户单位需维持一个国库，以存放剩余粮食、布匹、钱币及其他物品。两司马管理账目并逐级上报。每逢红白喜事等需要之时，由社区按固定标准从公共积蓄中支取：文本规定婚丧各拨钱一千文、粮一百斤，从公共供应中支出。[^tianmu][^michael-ii]

每个单位还须维持一座礼拜堂，所有居民每七日在此聚会，赞美天父，听诵圣经，并接受两司马的教导。儿童被教授太平天国经典和诏令。两司马集乡长、司祭、军需官、审判官和征兵官的角色于一身。[^luo-ergang]

文本还规定了对于弱势群体的赡养：鳏寡、孤儿、无子女老人、残疾者及病患由公共仓储供养。这一条款虽属笼统，却赋予田亩制度一种在清朝官方话语中不常见的福利维度。[^tianmu]

## 与军事组织的关联

田亩制度中的层级体系并非纯属民事。它建立在与太平军相同的组织矩阵之上。其链条自下而上为：

- 5户 → 伍长
- 25户 → 两司马
- 100户 → 卒长
- 500户 → 旅帅
- 2,500户 → 师帅
- 13,156户 → 军帅[^michael-ii][^luo-ergang]

这与军事指挥链条相对应，同样的职衔施用于统率士兵的军官。文本明确指出：太平之时，军官监理农耕；危难之际，同批军官率其部众出战。农家即基本军事单位。村庄即营寨。民政即处于休憩状态的军队。[^michael-ii]

这种民政与军事秩序的融合使该制度在纸面上颇为严密。但这也意味着，该制度只有在太平天国军事权威稳固有力的地方才能运作——而这一条件在少数核心区域之外极少能维持超过数年。

## 圣库理念

圣库既是一种制度，也是一种理念。田亩制度将其视为超出家庭生计之外全部剩余产品的汇集之所。物品不应为私人所积储。文本中的名句勾勒了这一愿景：

> 有田同耕，有飯同食，有衣同穿，有錢同使，無處不均勻，無人不飽暖。[^tianmu]

在实践中，圣库更多地作为一个军事供给系统而非全民仓储来运作。在广西时期，信众将财产交归公用基金。在征战期间，战利品和征发物经由圣库体系流转。在天京，圣库接收赋税、贡物和没收的财富，然后向官员、士兵及其家属分配口粮、衣物、武器和军需品。该制度作为战争金库运作良好；但它是否能转变为一个民事国库，则无从作答。[^luo-ergang]

## 实际发生了什么：实施及其限度

田亩制度作为土地重新分配方案从未被完全实施。不同地区的证据显示，文本规定与战时效用之间存在巨大落差。[^michael-ii]

在天京附近及长江下游沿岸地区——太平军军事存在最为强大的地方——太平天国确实颁发了田凭和税票。一些地方记载提及太平天国官员分配土地使用权。但九等田分类制度、系统性重新分配以及完整的二十五户单位结构的证据均十分薄弱。更为常见的是，太平天国允许原有地主继续留在其土地上，只要他们向太平天国当局缴税或交粮。在某些地方，地方强人和前清吏员继续充当中间人。[^kuhn]

在安徽及江西部分地区，太平天国有时颁发新税册并试图向耕种者直接征税。但这些努力参差不齐，常常被清军的军事行动打断，且很少能持久。在浙江——战争后期（1861–1862年）方才攻占——太平天国在清军反攻之前几乎来不及建立系统的行政管理。[^luo-ergang]

在不能或不曾重新分配土地的地方，太平天国一般依赖征税——通常是一种在收获时征收的粮食税，在某些地方比清税更重，在另一些地方则更轻。这远非文本中描绘的从公仓中按均等份额分配的愿景，但它为战争提供了收入。[^michael-ii]

## 来自不同地区的证据

罗尔纲依据地方志、太平天国文书和清朝报告，找到了太平天国在数省推行土地政策的零星证据。[^luo-ergang]

在江苏部分地区，尤其是常熟、无锡一带，太平天国官员颁发了田凭，承认耕种权并设定纳税义务。一些田凭直接署的是个体耕种者而非地主的姓名，表明太平天国试图绕过或取代原有地主阶级。但这一制度并不统一：在其他县，同样的田凭却发给了公认的地主。

在湖北，太平天国曾短暂尝试登记土地并征收直接税，但清军攻势打断了这一努力。在广西，早期拜上帝会社区曾实行一种形式的公共仓储，但这属于信众之间的志愿安排，而非政府行为。

迈克尔指出，19世纪50年代的外国观察者——包括英国驻上海领事官员——报告称，太平天国在征税，但似乎并未系统性地实施田亩制度中的重新分配条款。[^michael-iii]

## 争论：田亩制度是否被实施？

围绕这一问题的史学论争受到相互竞争的政治与学术议程的塑造。在中华人民共和国，尤其是在毛泽东时代，太平天国起义被颂扬为一场农民革命，田亩制度则被视为平等主义、准共产主义纲领的证据。罗尔纲——中国太平天国研究的泰斗——主张，尽管该制度未被完全实施，但它代表了一种真正的革命抱负，并在太平天国军事控制时期在有限区域内得到了应用。[^luo-ergang]

持怀疑态度的学者——包括迈克尔及更近期的学者——指出，从未有人提供过重新分配的全面证据，该文本或许是一份宣示性的文件而非行政计划，而战时条件使得系统性实施成为不可能。孔飞力对中国地方社会的分析表明，任何攻击私有土地所有制的纲领都会遭到地主和普通农民双方的强力抵抗——农民自身对土地也有着错综复杂的关系。[^kuhn][^michael-ii]

第三种观点，与更近期的中国学术研究相关，认为田亩制度应被理解为一篇政治宣示书——一份宣告太平天国意图并鼓舞士气的文本，即使它并未作为一份严格意义上的行政条令得到执行。在这一解读中，该文本之所以重要，在于它展现了太平天国的价值观，而不在于它作为政策取得了什么成就。[^reilly]

最为审慎的结论是：田亩制度是一半纲领、一半宣传；其中某些要素——尤其是税收、粮食征发和公共仓储——在占领区得到了应用；全面的重新分配方案从未被认真尝试；而无论其实际命运如何，该文本仍然是太平天国意识形态的一份非凡陈述。
