# 关键争议问题

| 争议问题 | 通俗解释 | 当前暂定立场 |
|---|---|---|
| 太平天国是否为基督教？ | 太平天国使用了源自基督教的文献和语言，但围绕洪秀全的主张建立了中国式宗教王国。Reilly 认为太平天国宗教是一种独特的中国融合体系——并非基督新教教会，亦非简单的基督教异端。与太平天国接触过的传教士（包括罗孝全和理雅各）看法分歧：有些人视其为虽真诚但被误导的基督教信仰，有些人则视其为亵渎。[^reilly] | 应将其视为太平天国宗教，而非基督新教教会。[^reilly] |
| 天朝田亩制度是否施行？ | 《天朝田亩制度》文本承诺重新分配土地和建立公共仓储。罗尔纲视其为原始共产主义倾向的例证；后来中华人民共和国学术界大体放弃了这一解读。Michael 及多数西方史家认为该制度从未系统实施——它更像一份意图宣告，而非行政现实。[^tianmu][^michael] | 方案文本清晰；其实施状况需逐地考证。[^tianmu] |
| 洪秀全是疯子还是谋略家？ | 各种史料既描述其异象与宗教执念，亦记述其政治领导力。部分西方记述（包括早期传教士报告）将洪的幻象视为妄想。中国史学在将其视为精神失常的宗教狂热者与理性的农民革命者之间反复摇摆。Reilly 认为医学问题无法回答，标签遮蔽多于揭示：关键在于洪的主张在太平天国体系内如何运作。[^reilly] | 避免使用医学标签；阐明其主张及其效果即可。[^reilly] |
| 太平天国为何失败？ | 内部冲突、战略过度扩张、清朝应变、地方团练武装、外国人支持的长江下游军事行动，以及供应困难等因素共同作用。Kuhn 强调向地方军事化转变的结构性因素，这一转变使清军在1860年后产生了质量上远超绿营的湘军与淮军——这是与太平天国战争初期被太平军击溃的绿营根本不同的军事体制。[^michael][^kuhn] | 多因素分阶段分析。[^michael][^kuhn] |
| 死了多少人？ | 估计在1000万至3000万之间，以2000万为常见概数。何炳棣对清代人口数据的重建提供了方法论基础，而 Meyer-Fong 则批评将大量伤亡数字视为自明事实而非建构产物的倾向。所有数据皆为基于人口学重建得出的宽泛估算，而非计数所得——战争年代不存在可靠的普查数据。[^ho-pingti][^meyer] | 概览中用宽泛估计，细节中用分区域分析方法。[^ho-pingti][^meyer] |
| 太平天国是否为原始共产主义者？ | 后世读者从其中读出了公共财产和土地重新分配的语言。20世纪50至70年代的中华人民共和国史学，承袭罗尔纲的早期研究，将太平天国定性为预示共产主义革命的农民革命运动——这一解读不仅源于原始史料，也受马克思主义历史分期的影响。西方史家以及1980年代后的中华人民共和国学者，一般视此为一种后世类比，而非直接谱系。[^luo] | 属后世类比，非直接谱系。 |
| 是否外国军队击败了太平天国？ | 外国统率的军队在上海及长江下游一带发挥作用。戈登的同代人传记作家 Wilson 将戈登的战役视为决定战局的关键。后来的学术研究——包括 Michael 和 Kuhn——认为外国军队固然重要，但相对于由曾国藩和李鸿章统领、承担了大部分战斗并控制了关键战场的中国地方军队而言，居于次要地位。[^wilson] | 在特定战场扮演重要角色，但不足以解释整个战争的结局。[^wilson] |
| 李秀成自述是否可靠？ | 李秀成《自述》是在1864年被俘至被处决的短短数周内，于清朝监禁中写成的。这是太平天国高级将领最重要的第一人称叙述。学者们争论该文本究竟反映了李秀成真实的声音，还是受到俘获者——已知曾国藩编辑过提交清廷的版本——的影响。文中自我开脱的口吻（指责洪秀全的战略失误）及其对个别清军将领表现出忠顺之意的表述，令一些人质疑部分内容是否写于胁迫之下或被后世篡改。[^michael] | 极具价值，但须与其他来源参照比对。 |
