# 详细年表

| 日期 | 事件 |
|---|---|
| 1814年 | 洪秀全生于广东花县。[^michael-i] |
| 1827年 | 洪秀全首次参加府试落第。[^michael-i] |
| 1836年 | 洪秀全再次落第；在广州收到梁发所赠基督教传道书，包括《劝世良言》，但并未阅读便搁置一旁。[^michael-i] |
| 1837年 | 洪秀全第三次应试落第后患重病，梦中见一位老人与一位中年人，后来他将此解释为上帝天父与耶稣基督。[^michael-i] |
| 1843年 | 洪秀全第四次落第。随后在表弟李敬芳催促下阅读梁发的传道书，通过基督教框架重新诠释了1837年的异象，并开始传道。[^michael-i] |
| 1844年 | 洪秀全与冯云山前往广西传道。洪不久返回广东；冯留在广西，开始在桂平县紫荆山一带组织拜上帝会。[^acs-jintian] |
| 1844—1847年 | 冯云山将拜上帝会发展为广西一个有数千人的组织化社群，吸引了客家、壮、瑶等族群信徒，并建立了领导架构。此为起义的组织基础。 |
| 1847年 | 洪秀全赴广西加入冯云山及拜上帝会。洪开始撰写核心教义文献（《原道救世歌》《原道醒世训》《原道觉世训》）。[^michael-i] |
| 1851年1月11日 | 拜上帝会众聚集金田，举旗起义。[^acs-jintian] |
| 1851年9月 | 太平军攻克永安，开始更全面的立国建制，即永安建制：洪秀全称天王，分封五王（杨秀清、萧朝贵、冯云山、韦昌辉、石达开）。[^acs-jintian] |
| 1852年4月 | 太平军突破永安之围，开始北上进军。[^acs-jintian] |
| 1852年5月 | 冯云山在北上途中战死于广西全州。[^luo-feng] |
| 1852年9月 | 萧朝贵在进攻湖南长沙时阵亡。[^luo-xiao] |
| 1852年12月 | 太平军攻克中游江畔的武汉三镇之一武昌。[^acs-jintian] |
| 1852年12月—1853年1月 | 太平军据守武昌，洗劫其府库军械，征募当地船民与劳工，为东下进军组建船队。 |
| 1853年2—3月 | 太平军沿长江而下，经九江、安庆直逼南京，沿途城池几乎未遇抵抗。[^acs-jintian] |
| 1853年3月19—20日 | 太平军攻克南京，洪秀全改其名为天京（天国之都）。此城成为此后整个战争期间太平天国的政权所在地。[^michael-i] |
| 1853—1855年 | 北伐：林凤祥、李开芳率军北上直逼北京。兵锋最远至天津城郊，后被击退、包围并歼灭。[^acs-jintian] |
| 1853—1856年 | 西征：太平军转战中游长江流域，攻取并争夺安庆、九江、南昌、武昌、岳州等地。此役使其与曾国藩麾下新兴湘军陷入持续交战。[^acs-jintian] |
| 1856年9—11月 | 天京事变：杨秀清奉洪秀全之命被韦昌辉所杀；韦昌辉随后屠戮杨的追随者（数千人遇害）。石达开返回天京谴责韦昌辉；韦图谋杀石，石出逃。洪随后命人诛杀韦。这场内部杀戮摧毁了太平天国领导核心，成为战争转折点。[^michael-i] |
| 1857年 | 石达开与洪秀全决裂后率太平军大部离开天京——洪日益猜忌石，并提拔自家兄长（洪仁发、洪仁达）以牵制石。石独立转战于江西、浙江、福建、湖南、广西、贵州、四川。 |
| 1858年11月 | 三河之战：陈玉成与李秀成在安徽三河全歼李续宾所率湘军一支部队。此为天京事变后太平军取得的最大战场胜利，暂时阻止了清军攻势。 |
| 1859年 | 洪仁玕在与堂兄洪秀全失散多年后抵达天京，受封干王，并颁布《资政新篇》——一份吸纳西方制度的改革纲领。[^zizheng] |
| 1860年 | 二破江南大营：李秀成率太平军摧毁南京城外的清军围城大营，暂时解围，为东征打开通道。 |
| 1860年 | 华尔组建洋枪队（后改称常胜军），以保卫上海抵御太平军压力。[^wilson] |
| 1860—1861年 | 安庆保卫战：曾国藩湘军围攻长江上南京战略门户安庆。陈玉成屡次率援军解围，均告失败。此役为后期战争的决定性战略对决。 |
| 1861年9月 | 安庆陷于湘军之手。此役切断了太平天国对中游长江的控制，打开了通往南京的江上通道。 |
| 1861年 | 据美国国会图书馆记录，《英杰归真》在南京刊印。[^yingjie] |
| 1861—1862年 | 李秀成东征：李率太平军穿浙江，克杭州、宁波及该省大部，同时威胁上海。此役将太平天国控制区向东扩展，但使其兵力过度分散。 |
| 1862年 | 陈玉成在安徽寿州遭苗沛霖出卖，被清军俘获。不久后被处决，年仅25岁。他的死使太平天国失去了最有能力的野战指挥官。[^acs-chen] |
| 1862年 | 华尔在浙江慈溪伤重而死。[^wilson] |
| 1862年 | 曾国藩湘军于本年开始包围南京，展开最后围攻。[^michael-i] |
| 1862—1863年 | 上海战役：李秀成屡次试图攻取上海，但遭到常胜军、英法正规军及李鸿章淮军的防守。太平军所有进攻均未成功。 |
| 1863年12月 | 苏州陷于李鸿章淮军及戈登指挥的常胜军之手。献城投降的八位太平天国将领被李鸿章下令处决——"苏州杀降"，此事导致戈登与李鸿章之间的决裂。 |
| 1864年6月1日 | 洪秀全死（一说服毒自尽，一说因病）于南京，时清军围城日紧。[^michael-i] |
| 1864年7月19日 | 清军攻占天京/南京。湘军洗劫焚毁全城。李秀成在试图护送洪秀全幼子洪天贵福出逃时，不久被俘。[^michael-i] |
| 1864—1866年 | 太平军余部在江西、福建、广东继续零星抵抗，包括李世贤与汪海洋所部。 |
| 1863年 | 石达开在四川大渡河被俘，于成都处决。 |
| 1860年代—1890年代 | 重建重塑了各受影响地区的地方社会：人口锐减区域的重新安置，儒家机构的修复，战争死难者的纪念祠，以及塑造战争记忆的地方史志编纂。[^meyer] |
| 20世纪 | 民国与中华人民共和国研究者将太平天国重新诠释为反清革命、农民战争或革命先驱。孙中山和毛泽东均将太平天国引为先驱。 |
| 1980年代至今 | 学术研究扩展到宗教、地方社会、记忆、军事历史及全球语境等领域。[^reilly][^meyer][^kuhn] |
