# 外国人物

太平天国内战发生于西方在华扩张的时代，来自外国的参与者——雇佣军、传教士、外交官、条约口岸商人、军官及记者——扮演了从外围观察者到战场直接统率者的多重角色。理解外国维度对于理解上海的战局结果、这场战争在十九世纪全球史中的地位，以及存世文献的性质而言，至关重要。外国参与可分为若干相互重叠的类别：传教士接触与宗教交流；领事观察与外交中立政策；私人军事企业（常胜军中的"外国军官"传统）；以及英文《北华捷报》上的条约口岸报章报道。

## 外国军事部队

除雇佣军官个人外，在某些战区还有成建制的外国军事力量运作：

- **1860—1862 年上海地区的英法部队：** 第二次鸦片战争（1856—1860 年）结束后，驻扎上海的英法正规部队与常胜军协作，防御该城免遭太平军攻击。其存在对防止太平军夺取该城起到了决定性作用。
- **1862 年宁波的法国防御：** 一支由日意格与德克碑统领的法国海军与地方清军混合部队，防御了宁波这一口岸免遭太平军进攻；此次交战标志着法军与清军在浙江开始积极军事合作。

这些军事干预主要由条约口岸商业利益驱动，而非出于对清朝的同情，且几乎完全集中在上海—宁波沿海地带。

## 中立问题

从 1853 年至约 1860 年，英、法、美三国的官方政策是在清朝和太平天国之间保持中立。这一政策受到以下因素的考验：（a）太平天国激进的禁烟立场，威胁了英国财政收入；（b）1860 年后太平军对条约口岸腹地发起的进攻；以及（c）清廷（在 1860 年《北京条约》之后）愿意接受外国军事援助。到 1862 年，中立已事实上崩溃并转为对清朝的积极支持。
