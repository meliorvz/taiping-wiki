# 伤亡估计及其解读方法

太平天国内战（1850—1864 年）被广泛视为十九世纪最为惨烈的冲突。然而其伤亡数字仍是中国近现代史中最受激烈争议的数据之一。战前不存在可与近代西方国家相匹敌的人口普查体系，战后又没有系统的伤亡统计，大量官方档案在战斗中被焚烧、在县级政府崩溃时遭劫掠，或被各敌对政权出于掩盖地方状况的目的而刻意销毁。清朝的奏折和地方志仅以碎片形式存世；传教士和条约口岸的记述提供了部分外部观察；而人口学重建则依赖税册、族谱及迁徙模式，而这些资料本身即不完整。[^meyer-fong]

其结果是，所有关于此战的伤亡数字均为估计值——而可信估计的范围根据输入人口学模型的假设条件和统计地理边界不同，从约 1000 万到 3000 万超额死亡不等。

## "2000 万"数字

最常被引用的总量——2000 万死亡——源于历史学家何炳棣（1959 年）的人口重建，他将道光（1820—1850 年）与同治（1862—1874 年）时期的清朝赋税登记及方志记录加以比较，发现在受战争影响最重的省份中约有 2000 万人口赤字。[^ho-pingti]

支持 2000 万估计者认为，该数字与以下事实相符：（a）何炳棣认识到江苏、安徽、浙江、江西和湖北的登记人口在战前战后登记册之间下降了数千万；（b）传教士及外国观察者关于长江走廊大规模人口消失的一致报告；以及（c）即使保守的县级研究通常也在直接交战区域发现 30—60% 的死亡率。

批评者则指出，该数字：（a）将人口迁移与死亡混为一谈——许多逃往上海、条约口岸或未登记内陆地区的人们只是从税册中消失；（b）依赖的是清代的登记数据，而这些数据甚至在战前即已长期不可靠；且（c）将省级赤字简单外推为一个整洁的整数，却未关注死亡的具体原因。[^meyer-fong]

其他估计在 1000 万（更保守，将战争区域外的间接饥荒效应排除在外）和 3000 万（包括捻军起义死亡、同期的回民起义以及整个世纪中叶危机期间的间接饥荒死亡）之间浮动。

## 各区域伤亡估计

受创最重的区域是太平军拥有领土且清朝地方军队在此进行长期围城作战的长江走廊诸省。

| 安徽 | 400—500 万 | 安庆围城、焦土作战、饥荒 | 湘军 1861 年围安庆；1853—1864 年连续劫掠 | 部分县登记人口损失超过 50%；方志证据[^luo] |
| 浙江 | 300—400 万 | 后期太平军攻势、清朝之反攻 | 杭州 1861 年被占；1864 年收复；1862 年宁波外国防御 | 杭州战前人口约 100 万，战后约 7 万（或有夸大，但足见其剧烈） |

| 湖北 | 200—300 万 | 武昌战役、长江走廊作战 | 武昌三度易主（1853、1854、1855 年）；连续不断作战 | 武昌/武汉反复易手；城市人口遭到毁灭性打击 |
| 湖南 | 50—150 万 | 兵员抽调、捻军溢出效应、地方暴力 | 湘军招募抽走大量成年男性；长沙 1852 年防守成功 | 直接战斗较少，但因军事招募大量流失人口[^kuhn] |

| 广东 | 50—100 万 | 天地会战乱、难民危机、招募/镇压 | 天地会起义；洪秀全故乡所在的县 | 与太平天国主要战争相关但独立存在；土客械斗 |

| 河南/直隶 | 30—100 万 | 北伐行军路线 | 太平军北伐 1853—1855 年；后期捻军战役 | 北伐军于天津附近被全歼；沿途村庄被焚毁 |

## 军事伤亡

**太平军：** 没有存世的全面名册。鼎盛时太平军野战兵力约 300,000—500,000 人；含随营人员及家属则大体在 500,000—600,000 人（罗尔纲）。若干具体战役的最佳估计：
- 南京围城（1862—1864 年）：可能 100,000 名以上太平军守城者在最后进攻及随后的大规模屠杀中丧生
- 安庆（1861 年）：该城陷于湘军之手时约 16,000 名太平军守军被屠
- 苏州（1863 年）：围城期间数万人死亡，以及随后李鸿章违背戈登抗议下令处决已投降太平军将领（"苏州杀降"）
- 北伐（1853—1855 年）：整个太平军北伐部队约 70,000—80,000 人被全歼

**清朝正规军（绿营与八旗）：** 绿营与八旗在战争早期（1850—1856 年）遭受了重大损失，包括武昌和南京数支八旗驻军的全军覆没。南京八旗驻军于 1853 年被屠。清朝正规军自 1856 年以后逐渐被地方军队取代，其作战死亡总数估计在 200,000—400,000 人之间，但记录零散不完整。[^kuhn]

**湘军：** 曾国藩之湖南武装力量在整场战争中估计损失了 50,000—100,000 人，损失最重的是三河之战（1858 年），李续宾所率 6,000 人几乎被全歼。

**淮军：** 李鸿章之安徽武装力量，主要运用于长江下游和上海战区，可能损失了 10,000—20,000 人。

## 方法论：当档案被销毁，历史学家如何推估死亡人数

学者们采用若干种互相叠合的技术：
1. **人口登记对比法：** 比较战前战后税册（丁口册）和方志人口统计，以估计净损失。弱点：税册统计的是纳税人，而非实际人口；人口迁徙和逃税歪曲了数字。
2. **族谱分析法：** 利用家族谱系（族谱）追踪战前战后世代之间消失的家族分支。弱点：只有存续下来的家族才留有谱牒。
3. **同时代观察者记述法：** 传教士信函、领事公文以及《北华捷报》报道描述了特定地方的情形。弱点：观察者能接触的范围有限，且倾向于报道最惨重的情况。
4. **考古与物质证据法：** Meyer-Fong 的方法——阅读"遗存之物"——墓葬、石碑、纪念碑铭文、地方记忆文本——作为损失规模和分布的佐证。[^meyer-fong]
5. **粮价与饥荒指标外推法：** 使用价格数据和气象记录来模拟饥荒严重程度及预期死亡率。

现代学者（Meyer-Fong、Michael、罗尔纲、Kuhn）的共识是：没有任何一个单一数字能完全反映这场战争的总体人员代价，最好的方法是将宽泛的总体估计与承认不确定性的详细区域证据结合起来。[^michael][^luo][^kuhn]
