# 主要受影响区域

太平天国内战并非均等地影响中国全境。其战祸集中在长江走廊——从南部广西贯穿湖南、湖北、江西、安徽，直至长江下游的江苏和浙江——而次要战区则位于广东、福建和华北平原。理解战争的地理分布对于把握其人口、经济和政治后果，以及评估因地而异、差异悬殊的死伤估计至关重要。

可将战争视为三个同心区域：**太平天国核心区**（天京及周边被持续控制数年的府县）、**争夺走廊**（反复易手并遭受最重破坏的县份），以及**外围袭扰区**（太平军远征所及但从未稳固控制的区域）。这些区域的经济地理位置——为清朝国家提供粮食、盐、丝绸和茶叶的岁入——在很大程度上决定了战争的战略逻辑及其财政后果。

## 地理特征

冲突在每一阶段都深受地理条件塑造：
- **长江走廊：** 战争的主轴。太平军于 1853 年沿江自武昌东下南京，清军反攻则沿相反方向逆江而上。控制长江即意味着控制部队调动、物资供应和通信联络。
- **鄱阳湖：** 为太平军与湘军水师之间大规模水战的发生地（1854—1858 年）。控制该湖即决定了江西的归属。
- **大运河：** 传统的南北粮食运输大动脉因太平军占领镇江和扬州而中断，迫使清朝将粮食运输转为海运——这一永久性转变带来了深远的财政后果。

## 经济地理

战争的财政维度与区域经济地理密不可分：
- **江南——丝绸、棉花、茶叶、余粮区：** 帝国最富庶的地区，故为双方争夺的宝物。江南府县的赋税收入为清朝国家提供财力保障；太平天国占据此区域对北京而言是一场财政大灾难。
- **湖南——粮食与兵员：** 湖南所受战争破坏较长江下游更轻，故成为了湘军的主要物资和兵员基地。
- **盐区：** 位于苏北/皖北的两淮盐区是清朝收入的重要来源。太平军与淮军在此区域反复争夺，扰乱了帝国食盐财政。
- **条约口岸带——上海、宁波：** 由外国管理的口岸海关收入为清廷提供了战后最可靠的财政收入来源，并激励了外国以军事合作确保口岸不落入太平军之手。
